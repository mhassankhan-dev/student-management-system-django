# Student Management System (Django)

A simple CRUD-based Student Management System built using Django.

---

## 🚀 Features

- Add Student
- View Students
- Edit Student
- Delete Student
- Bootstrap UI
- Django Admin Panel
- ModelForm based validation

---

## 🛠 Tech Stack

- Python
- Django
- SQLite
- Bootstrap 5

---

## 📂 Project Structure

- config/ → project settings
- students/ → main app
- templates/ → HTML files
- models.py → database schema
- views.py → logic

---

## ⚙️ Setup Instructions

```bash
git clone https://github.com/mhassankhan-dev/student-management-system-django.git
cd student-management-system-django
python -m venv venv
venv\Scripts\activate
pip install django
python manage.py migrate
python manage.py runserver
